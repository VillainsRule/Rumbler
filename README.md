<h2 align="center">Installation</h2>

1. Clone the repo from Git:
```bash
git clone https://github.com/VillainsRule/Rumbler.git && cd Rumbler
```
2. Install dependencies:
```bash
npm i
```
3. Edit values in `config.js`. Add tokens to `tokens.txt`.
4. Run the program!
```bash
node .
```
